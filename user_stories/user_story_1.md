# User Story #1

## As a shipping platform manager, 
## I want to configure following input types:
* eu-parcel-to-tuscany
* eu-letter-to-sicily

## So that the following shipping arrangements are created on the platform
### Storage booking
- type eu-parcel-to-tuscany ? large_storage_booking
- type eu-letter-to-sicily ? small_storage_booking
### Transportation booking
- type eu-parcel-to-tuscany ? truck_transportation_booking
- type eu-letter-to-sicily ? ship_transportation_booking
